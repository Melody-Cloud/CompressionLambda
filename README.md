# Compression lambda

This is code for "compression lambda" used in "Melody Cloud" project. 

Its purpose is to run ffmpeg commands inside AWS Lambda runtime.

The code uses ffmpeg binary, so it needs to be deployed manually. All assets, that need to be bundled with code in deployment package could be found here:
